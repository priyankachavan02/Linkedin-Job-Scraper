# Linkedin-Job-Scraper

**[Live Demo](https://drive.google.com/file/d/1MkwLnk8H_q0PzNJ9Ruruzu1Oq7YuX7ii/view?usp=drive_link)**

---
A Python-based automation tool that scrapes **LinkedIn job listings** based on keywords, location, and filters, and saves them in a structured format for easy analysis.  
Built using **Selenium**, **BeautifulSoup**, and **Pandas**, it helps job seekers analyze trends, identify hiring companies, and collect data for ATS optimization.


## Features

 **Automated Job Search**
- Logs into LinkedIn and performs job searches for specific keywords or locations.
- Extracts key job details: title, company, location, experience level, and post date.

 **Data Extraction & Storage**
- Parses job information using **BeautifulSoup**.
- Saves structured job data into **CSV or Excel** files.

 **Smart Delays & Handling**
- Uses random sleep intervals to mimic human behavior.
- Avoids LinkedIn’s anti-bot detection mechanisms.

---

## Tech Stack

**Language:** Python  
**Libraries & Tools:**
- `Selenium` → for browser automation  
- `BeautifulSoup4` → for HTML parsing  
- `Pandas` → for tabular data storage and export  
- `python-dotenv` → for handling credentials securely  
- `time`, `random` → for delay simulation
